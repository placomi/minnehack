# MNHack26 Web Application

## Overview
This is our submission for a MNHack26. The app hosts crowd-sourced data regarding protests, protests, riots, and conflicts across the world. It's populated with data from webscraped tweets. Users can also add "Snippets" on the website itself.

## Features
- User authentication
- Heatmap of protests, riots, and conflicts
- Field to input event data.

## Setup Instructions

1. Clone the repository:
   ```
   git clone placom/mnhack26
   ```
2. Start Next.js
   ```
   cd web
   npm i
   npm run create-db (You must have docker installed)
   npm run start
   ```
3. Start scraper
   ```
   cd scraper
   tbd
   ```
